# Select 选择框

当选项过多时，使用下拉菜单展示并选择内容。

## Default

设置 `type`、`color` 属性来改变 Select 的颜色（`color`属性覆盖`type`属性）

`type`: primary | warning | success | danger | light | dark

`color`: hsl | rgb | hex
