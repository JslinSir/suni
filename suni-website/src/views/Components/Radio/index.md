# Radio 单选框

在一组备选项中进行单选

## Default

设置 `type`、`color` 属性来改变 Radio 的颜色（`color`属性覆盖`type`属性）

`type`: primary | warning | success | danger | light | dark

`color`: hsl | rgb | hex
