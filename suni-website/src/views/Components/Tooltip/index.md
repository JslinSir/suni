# Tooltip 提示框

常用于展示鼠标 hover 时的提示信息。

## Default

设置 `position` 属性来改变 Tooltip 的位置

`position`: left | top | bottom | right

```vue demo src="./default.vue"

```

## Type & Color

设置 `type`、`color` 属性来改变 Tooltip 的颜色（`color`属性覆盖`type`属性）

`type`: primary | warning | success | danger | light | dark

`color`: hsl | rgb | hex

```vue demo src="./type.vue"

```

## Variant

设置 `variant` 属性来改变 Tooltip 的风格

`variant`: default | border | border-thick | shadow

```vue demo src="./variant.vue"

```

## Circle & Square

添加 `circle` / `square` 属性来改变 Tooltip 的形状

```vue demo src="./circle.vue"

```

## Loading

添加 `loading` 属性来切换 Tooltip 的状态

